@Tags em HTML

-> <html> </html> -> Tag principal 
-> <head> </head> -> Área de configurações
-> <body> </body> -> Área do corpo
-> <h1> </h1> -> Header
-> <title> </title> -> Título do site (Aba)
-> <hr> -> Cria linha horizontal
-> <p> </p>-> Parágrafo
-> <meta> -> Define algumas configurações através de parâmetros
    -> charset
    -> content
